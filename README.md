# Google Sheet AI Analyzer (Frontend)

A lightweight React/TypeScript interface built as an internal tooling proof-of-concept.
It allows users to link a public Google Sheet and query the underlying data using natural language.

### Links

Live link: https://google-sheet-analyzer-frontend.vercel.app/
Backend repo: https://github.com/alexandercsierra/google-sheet-analyzer-backend

### Tech Stack

- React, TypeScript
- UI Focus: Minimalist, functional proof-of-concept layout
